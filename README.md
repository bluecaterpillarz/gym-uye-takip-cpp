# gym-uye-takip-cpp
# Spor Salonu Üye Takip Sistemi (C++)

A console-based C++ application for managing gym member records.

Bu proje, spor salonu üyelerinin bilgilerini ve vücut ölçümlerini
takip etmek için geliştirilmiş bir C++ konsol uygulamasıdır.

## Özellikler
- Yeni üye kaydı oluşturma
- Mevcut üyeler için CSV dosyasına yeni kayıt ekleme
- VKİ (Vücut Kitle İndeksi) hesaplama
- Yaş ve cinsiyete göre VKİ değerlendirmesi
- Yağ oranı değerlendirmesi
- Dosya okuma ve yazma (CSV)

## Teknik Notlar
- Proje fonksiyonel (procedural) C++ yaklaşımıyla geliştirilmiştir
- Dosya I/O, veri doğrulama ve kontrol akışı odaklıdır
- Konsol tabanlıdır, ek kütüphane gerektirmez

## Çalıştırma
C++ destekleyen herhangi bir derleyici ile `main.cpp` dosyasını
derleyip çalıştırabilirsiniz.
